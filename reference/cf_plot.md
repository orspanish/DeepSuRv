# Plot original vs. counterfactual survival curves

Simple visualization comparing the original and counterfactual survival
curves.

## Usage

``` r
cf_plot(cf_out, main = "Counterfactual")
```

## Arguments

- cf_out:

  A list returned by cf_compare_one().

- main:

  Character; plot title.

## Value

A base R plot displaying survival curves.
